# group 1 project
 This project is about asking student to fill in their details so that they can have access to get in class.
 First it will ask the student to fill in their names, student number and password .
 Password will be their name and last four numbers of their student number .
 This is more like check in as it also record who attend the class and who did not.
 We have a class that contain properties of a student .
 We have function that generate password ( student name + the last four digit of student number ).
 We have funtion that validate a password ( one must make sure that the first password you enter matches with the last attemt).
 We have if else statement that grant acess if the password is correct and deny access if the password is incorrect .
 We that method that display the student's informtion..
 In the main program we asked the student to enter their information.
 We have instant of a class.
 We call out the methods. 
 We have while loop so that if the student enter the password that does not match the first attempt it will ask the user to enter the password.
 In conclusion the program ensure that relavent people enter the class and it keeps track of attendace.
